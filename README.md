「スカイルート」 
 
ゲーム概要 

巨大な世界樹を登り続ける 
2D縦スクロールアクションゲーム 

プレイヤーは探検家となり、 
崩れる足場や敵を避けながら、 
世界樹の頂上を目指す。 

ジャンル 

2D縦スクロール逃走アクション 

操作方法（予定） 

← → ：移動  

Space：ジャンプ 

 

 

ゲームの特徴 

① 崩れる足場 

木の枝や足場は、 
プレイヤーが触れることで徐々に崩壊する。 

そのため、 
同じ場所に留まり続けることができない。 

② 敵を倒せない 

本作では敵への攻撃手段は存在しない。 

敵の動きを見極め、 
ジャンプや移動で回避しながら進む。
